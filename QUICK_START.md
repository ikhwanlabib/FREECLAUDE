## 🎓 How to Use

### Basic Chat
1. Type a message in the input box
2. Press `Enter` to send
3. Claude responds with full context
4. Continue chatting (history maintained)

### Upload Files
1. Click 📎 button
2. Select file (.txt, .pdf, .py, .js, .md, .json, .csv, or image)
3. File content included in your message context
4. Claude analyzes and responds

### Switch Models
1. Click dropdown in top-right
2. Choose: Opus 4.7 / Sonnet 4.6 / Haiku / Opus Fast
3. Send next message with new model

### View Settings
1. Click ⚙️ button
2. Customize system prompt
3. Toggle streaming responses
4. Toggle code highlighting

### Get Help
1. Click ❓ button
2. View keyboard shortcuts
3. Learn about features
4. Check supported file formats

## 💡 Pro Tips

### For Best Results

1. **Be specific** in your questions
2. **Provide context** - more details = better answers
3. **Use files** to upload documents for analysis
4. **Ask follow-ups** to refine responses
5. **Try different models** - Opus is smarter but slower

### Model Recommendations

- **Simple questions** → Haiku 4.5 (fastest)
- **Coding help** → Sonnet 4.6 (balanced)
- **Complex analysis** → Opus 4.7 (smartest)
- **Time-sensitive** → Opus 4.6 Fast

### Save Conversations

1. Conversations auto-save to browser
2. They appear in sidebar as "Previous Chats"
3. Click any chat to resume
4. Or click "New Chat" to start fresh

## 🆘 Troubleshooting

**Q: Chat not working?**
A: Refresh page, check internet, try different model

**Q: File not uploading?**
A: File size too large, or unsupported format. Max 1MB recommended.

**Q: Chat history gone?**
A: Browser storage was cleared. Consider backing up important chats.

**Q: Slow responses?**
A: Try Haiku model for faster responses, or check internet speed.

**Q: How is this free?**
A: Puter.js "User-Pays" model - users cover their own Claude API costs.
